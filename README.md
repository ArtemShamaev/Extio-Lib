# extio - Расширенный ввод-вывод для Python

Библиотека для удобной работы с консолью: цветной вывод, ожидание нажатия клавиш и ASCII-персонажи.

## Возможности

- 🎨 **Цветной вывод** - 9 предустановленных цветов
- ⌨️ **Ожидание клавиш** - поддержка 25+ клавиш (включая F1-F12, стрелки, модификаторы)
- 🐧 **ASCII-арт** - 8 встроенных персонажей (Tux, Dog, Cat, Cow и др.)
- 🌍 **Кроссплатформенность** - Windows, Linux, macOS
